# Create-and-Cultivate-Aeronautics
A Minecraft modpack combining Create, Applied Energistics 2, Farmer's Delight, and Exploration mods — focused on aerospace engineering and automation. 

一个融合机械动力（Create）、应用能源2（AE2）、农夫乐事（Farmer's Delight）和探险模组（Exploration）的 Minecraft 整合包，主打航空航天工程与自动化。

## 🚀 快速开始
1.  下载本页面最新的 `.zip` 或 `.mrpack` 发布文件。
2.  在您喜欢的启动器（如HMCL、PCL、CurseForge App）中，选择“导入整合包”。
3.  选择下载的文件，等待安装完成即可启动游戏。
